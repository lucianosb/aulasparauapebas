Aula 02

Na aula 02 aprendemos um básico de HTML (Hyper Text Markup Language) usando para exercício prático o endereço html.lucianosb.com.br. Aprendemos como funcionam as estruturas de links, imagens, paragráfos e outros códigos. Vamos a breve revisão.

O HTML é formado por uma simples estrutura de marcação que determina os diferentes objetos de uma página da internet. Estas marcação são chamadas de tags e sua estrtura segue quase sempre o seguinte formato:


 CONTEUDO

Todo objeto HTML é chamado de tag e cada tag pode receber parametros que as fazem ter ações especiais. A tag de link é um desses exemplos como vemos abaixo.

Tag de links

A tag de links é uma das mais básicas da internet e representada pela letra a de "âncora". Seu principal parâmetro é o href que indica para qual endereço a pessoa irá após clicar no link.



  
  google.com

Observe que o espaçamento de linhas usados no exemplo acima não é obrigatório, escrevemos desse jeito apenas para facilitar a leitura. Poderíamos escrever tudo em uma linha só.


google.com

A quebra de linha no código, portanto, não representa a quebra de linha no resultado final. Para isso, usamos a tag de parágrafo.

Tag de Imagem

Esta é uma exceção a regra porque não precisa de indicação de fechamento.

A tag img irá produzir uma imagem bastando para isso que se indique uma fonte (source, abreviado como src) de onde está a imagem na internet através do parâmetro src.

Podem ser exibidas como imagem nessa tag os principais formatos de imagem: jpeg, jpg, gif, png.
